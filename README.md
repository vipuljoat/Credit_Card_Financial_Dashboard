# Credit_Card_Financial_Dashboard
Power Bi Dashboard

project Objective= To analyze credit card customer spending patterns and revenue contribution across different demographics, income groups, and regions to help the company improve targeted marketing and revenue strategies.

insights= High-income and Businessmen customers generate most revenue.
Age groups 30–40 and 60+ spend the highest.

steps=Imported data → cleaned → created visuals → added filters → final dashboard.
Used bar, line, and category breakdown charts to show trends.

action items= Offer premium rewards to high-income customers.
Run marketing campaigns in best-performing states.
